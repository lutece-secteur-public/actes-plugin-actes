
#Descriptif du plugin ACTES

##Introduction

Ce plugin permet aux collectivités locales de soumettre des actes (délibérations, arrêtés, ...) au contrôle de légalité en utilisant un dispositifde télétransmission et les normes d'échange définies par le MIAT (Ministère de l'Intérieur et de l'Aménagement du Territoire).

Ce plugin ne réalise pas lui-même la télétransmission vers le MIAT. Il fournit une interface (API) pour créer des modules qui vont s'appuyer sur les API des plates-formes de télétransmissionhomologuées par le MIAT. Les échanges entre le plugin et les modules reposent sur les structures issues du schéma XML de référence du protocole.

##Fonctionnalités du plugin

Les fonctionnalités proposées par ce plugin sont :


 
* Envoi d'un document accompagné éventuellement d'une ou plusieurs pièces jointes
* Consultation de l'état des télétransmissions réalisées
